Express Entry manages applications for 3 immigration programs: Federal Skilled Worker Program Federal Skilled Trades Program Canadian Experience Class Select a program above to see detailed requirements, or Compare all programs If you’re eligible for one of the above programs, you can also apply through Express Entry for the Provincial Nominee Program. If you’re nominated, you’ll get extra points so you can be invited to apply quickly.
